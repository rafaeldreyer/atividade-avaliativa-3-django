# Atividade Avaliativa 3 – Django e MySQL

Projeto desenvolvido para a Atividade Avaliativa 3 da disciplina, utilizando Python, Django e MySQL.

## Descrição
O sistema implementa um CRUD completo de funcionários, permitindo:
- Cadastro de funcionários
- Listagem de funcionários
- Edição de dados
- Visualização individual
- Remoção de registros

Os dados são persistidos em um banco de dados MySQL.

## Tecnologias Utilizadas
- Python 3.11
- Django 5
- MySQL
- Bootstrap 5
- Git e GitHub
